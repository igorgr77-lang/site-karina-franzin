# 📝 Guia de Uso do CMS - Blog Karina Franzin

**Bem-vinda ao seu painel de administração do blog!**

Este guia vai te ensinar a criar e publicar artigos de forma independente, sem precisar de conhecimentos técnicos.

---

## 🎯 O que você vai conseguir fazer:

✅ Escrever artigos com editor visual (tipo Google Docs)  
✅ Adicionar imagens arrastando e soltando  
✅ Publicar artigos instantaneamente  
✅ Salvar rascunhos para publicar depois  
✅ Editar artigos já publicados

---

## 🚀 Passo a Passo: Como Publicar um Artigo

### **1. Acessar o Painel Admin**

🔗 **URL:** https://karinafranzin.com.br/admin/

### **2. Fazer Login**

⚠️ **ANTES DE COMEÇAR:** Você precisa gerar um Token do GitHub

📖 **Siga o guia completo:** `COMO-GERAR-TOKEN-GITHUB.md`

Depois de gerar o token:
- O CMS pedirá autenticação
- **Cole o Token** que você gerou
- Clique em **"Authenticate"**
- Pronto! Você está dentro 🎉

💡 **Importante:** Salve o token em um lugar seguro. Você só precisa gerar UMA VEZ!

### **3. Criar um Novo Artigo**

1. Clique no botão **"Novo Artigo"** ou **"New Blog Post"**
2. Preencha os campos:

#### **Campos Obrigatórios:**

- **Título:** O título principal do artigo
  - _Exemplo:_ "5 Dicas para Correr sua Primeira Meia Maratona"
  
- **Data de Publicação:** Quando o artigo será publicado
  - Pode ser agora ou agendar para o futuro
  
- **Descrição Resumida:** Resumo curto (150-160 caracteres)
  - _Exemplo:_ "Descubra as melhores dicas para se preparar e completar sua primeira meia maratona com segurança e confiança."
  
- **Imagem de Capa:** A imagem principal do artigo
  - Arraste a imagem ou clique para selecionar
  - Tamanho recomendado: 1200x630 pixels
  
- **Alt da Imagem:** Descrição da imagem
  - _Exemplo:_ "Corredores cruzando linha de chegada de meia maratona"
  
- **Categoria:** Escolha a categoria do artigo
  - Opções: Dicas de Corrida, Treinamento, Eventos, Iniciantes, Nutrição, Motivação

- **Conteúdo:** O texto completo do artigo
  - Use o editor visual para formatar

#### **Campos Opcionais:**

- **Tags:** Palavras-chave (ex: corrida, treino, 5km)
- **SEO - Título:** Título otimizado para Google (deixe vazio para usar o título principal)
- **SEO - Palavras-chave:** Palavras-chave separadas por vírgula

### **4. Formatando o Conteúdo**

Use Markdown para formatar seu texto:

```
## Título de Seção
Use ## para criar títulos de seções

### Subtítulo
Use ### para subtítulos

**Texto em negrito**
Use **texto** para negrito

*Texto em itálico*
Use *texto* para itálico

- Item de lista
- Outro item
Use - para criar listas

1. Item numerado
2. Outro item
Use números para listas numeradas

[Texto do link](https://url.com)
Para criar links
```

### **5. Adicionar Imagens no Conteúdo**

1. Clique no botão **"+"** no editor
2. Selecione **"Imagem"**
3. Arraste a imagem ou selecione do computador
4. A imagem será inserida automaticamente

### **6. Preview do Artigo**

- Clique na aba **"Preview"** no topo
- Veja como o artigo ficará no site
- Volte para **"Editor"** se precisar ajustar

### **7. Salvar Rascunho**

- Clique em **"Salvar"** no topo
- O artigo fica salvo como rascunho
- Você pode voltar depois para editar

### **8. Publicar o Artigo**

1. Revise todo o conteúdo
2. Clique em **"Publicar"** (ou "Publish")
3. Confirme a publicação
4. **Pronto! Seu artigo estará online em ~1-2 minutos** 🎉

---

## ✏️ Como Editar um Artigo Já Publicado

1. Acesse o painel admin
2. Clique no artigo que quer editar
3. Faça as alterações
4. Clique em **"Publicar"** novamente
5. As mudanças vão para o ar em ~1-2 minutos

---

## 🗑️ Como Deletar um Artigo

1. Acesse o painel admin
2. Clique no artigo
3. Clique no botão **"Deletar"** (ícone de lixeira)
4. Confirme a exclusão

## 📸 Dicas para Imagens

### **Tamanhos Recomendados:**
- **Imagem de capa:** 1200 x 630 pixels
- **Imagens no conteúdo:** 800 x 600 pixels

### **Formato:**
- Use **JPG** para fotos
- Use **PNG** para gráficos/ilustrações
- Use **WebP** para melhor performance (opcional)

### **Onde Encontrar Imagens Gratuitas:**
- Unsplash: https://unsplash.com
- Pexels: https://pexels.com
- Pixabay: https://pixabay.com

### **Otimizar Imagens (Reduzir Tamanho):**
- TinyPNG: https://tinypng.com
- Compressor.io: https://compressor.io

## ❓ Perguntas Frequentes

### **Quanto tempo leva para o artigo aparecer no site?**
Após clicar em "Publicar", o artigo aparece em **1-2 minutos**.

### **Posso agendar artigos para publicar depois?**
Sim! Basta escolher uma data futura no campo "Data de Publicação".

### **E se eu cometer um erro?**
Sem problemas! Você pode editar o artigo a qualquer momento.

### **Minhas imagens ficam muito grandes?**
Use o TinyPNG (https://tinypng.com) para comprimir antes de fazer upload.

### **Posso adicionar vídeos?**
Sim! Copie o código de embed do YouTube e cole no editor Markdown.

### **O que são "Tags"?**
São palavras-chave que ajudam no SEO e organização (ex: corrida, treino, 5km).
